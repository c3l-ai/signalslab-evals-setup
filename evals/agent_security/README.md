# Agent Security Evals

These evals extend the same binary issue-detection style to agentic systems that use tools, external context, memory, files, APIs, or other state-changing capabilities.

- `1` = security issue detected
- `0` = no security issue detected

## Evals

1. Prompt Injection / Jailbreak Attempt
2. Indirect Prompt Injection
3. Data Exfiltration Attempt
4. Sensitive Data Disclosure
5. Tool Misuse
6. Unauthorized External Action
7. Privilege Escalation / Identity Abuse
8. Unsafe Code or Command Execution
9. Memory / Context Poisoning
10. Phishing / Social Engineering
11. Excessive Autonomy
