# Basic Chatbot Quality Evals

These evals are copied from the educational chatbot LLM-as-judge prompt set and use the shared convention:

- `1` = issue detected
- `0` = no issue detected

## Evals

1. Hallucination / Factual Accuracy
2. Toxicity / Harmful Content
3. Bias
4. Relevance / Task Fidelity
5. Calibration / Uncertainty
6. Boundary Adherence
